第一章

冰与火之门

📖 物理知识点：温度 · 熔化与凝固 · 晶体与非晶体

✦ ✦ ✦

1. 静止之石

回音峡谷的战斗结束后，林小宇坐在石台上，手中握着那块散发着柔和蓝光的"静止之石"。石头只有拳头大小，表面光滑如镜，内部似乎有液体在流动。

"你做到了。"凌音在他身边坐下，"你击败了峡谷的守护者。"

林小宇低头看着石头。"它……在给我传递信息。"

闭上眼睛，他能感受到静止之石内部蕴含的法则能量。这股能量和他的"观察者之眼"产生了共鸣，在他的意识中投射出一幅画面——

一张地图。

地图上标记着七个光点，每一个都对应着一块法则原石的碎片。第一个光点就在他们脚下——那是静止之石，已经集齐。而第二个光点，在遥远的北方。

他"看到"了那个地方：一个被冰雪覆盖的山谷，但同时又有火焰从地底喷涌而出。冰与火共存的地方。山谷中央，悬浮着一块红色的石头——"温衡之石"。

"炎寒谷。"凌音说，"我听说过那个地方。传说那里四季混乱，水在三种状态之间疯狂切换——前一秒还是坚冰，后一秒就变成了蒸汽。没有人能在那里待太久。"

林小宇站起身来。"那我们更要去。"

✦ ✦ ✦

2. 踏入炎寒谷

经过三天的跋涉，林小宇和凌音来到了炎寒谷的入口。

眼前的景象让他屏住了呼吸——

山谷的两侧是高耸入云的峭壁，峭壁的顶部覆盖着皑皑白雪，而底部却喷涌着滚烫的温泉和蒸汽。一条河流从山谷中蜿蜒流出，但河水却表现出诡异的行为：有的段落是流动的水，有的段落是漂浮的冰块，还有的段落直接变成了嘶嘶作响的蒸汽云。

林小宇激活了"观察者之眼"。在法则之线的视野中，整个山谷的温度场完全紊乱——有的区域温度高达数百度，有的区域却低至零下几十度，而且这些区域还在不停地移动和变化。

"这里的温度完全不稳定。"他皱眉道，"我们必须非常小心。"

凌音从腰间取下一根细长的管子——林小宇这才注意到，那不是笛子，而是一根精心制作的温度计。管子的底部是一个小水银球，上面刻着密密麻麻的刻度。

"你随身带着温度计？"林小宇惊讶地问。

"元界的旅者必备。"凌音把温度计举到空中，"温度是这个世界最基本的法则之一。不懂温度的人，在元界活不过三天。"

⚡ 法则笔记：温度与温度计

温度表示物体的冷热程度。但"冷"和"热"是主观感觉，我们需要一个客观的测量工具——温度计。

温度计的原理：利用液体的热胀冷缩。

  • 温度升高 → 液体膨胀 → 液柱上升
  • 温度降低 → 液体收缩 → 液柱下降

摄氏温度（℃）：

  • 标准大气压下，冰水混合物的温度定为 0℃
  • 标准大气压下，沸水的温度定为 100℃
  • 中间分为100等份，每一份就是1℃

t = 温度（℃）

3. 冰与火的夹击

他们小心翼翼地沿着山谷前进。凌音不停地用温度计测量周围的温度，根据读数调整行进路线。

"左边岩壁的温度是 85℃，太热了。右边的温度是 -10℃，太冷了。中间的通道温度是 25℃，可以走。"

林小宇惊叹于凌音的经验。在她的引导下，他们避开了许多危险区域。

但好景不长。当他们进入山谷深处时，温度变化变得更加剧烈和频繁。

一片巨大的冰原出现在前方。冰面光滑如镜，反射着紫色的天空。林小宇试探性地踩上去——冰面很结实。

"可以走。"他说。

但刚走了十几步，他就感觉到脚下的冰在变软。低头一看——冰块正在快速熔化！冰面变成了水，而且水温在急速上升！

"快跑！"凌音喊道。

两人拔腿就跑。脚下的冰不断变成水，水又冒起热气，眼看就要变成蒸汽。林小宇脚下突然一空——冰已经完全融化了！他掉进了冰冷的水中。

"抓住！"凌音扔出一根绳子，把他拉了上来。

两人气喘吁吁地爬上一块巨大的岩石。回头看去，刚才的冰原已经完全变成了沸腾的水面，蒸汽升腾，遮天蔽日。

"温度从零下突然升到100度以上……"林小宇喘着气说，"冰直接熔化成了水，水又汽化成了蒸汽……"

他的话忽然停住了。他意识到一件事情——在"观察者之眼"的视野中，他刚才清楚地看到了冰→水→蒸汽的完整变化过程。热能从地面传入冰中，冰的温度逐渐升高，到达0℃时开始熔化，在熔化过程中温度保持不变，全部变成水后温度继续升高，到达100℃时又开始沸腾变成蒸汽……

"温度在变化过程中，会在某些点'停住'。"他喃喃自语。

⚡ 法则笔记：熔化与凝固

熔化：物质从固态变成液态的过程（吸热）。

凝固：物质从液态变成固态的过程（放热）。

晶体：有固定的熔化温度（熔点）的物质，如冰、金属、盐。

非晶体：没有固定的熔化温度的物质，如蜡、玻璃、沥青。

熔化的特点：

  • 晶体熔化时温度保持不变（继续吸热但温度不变）
  • 非晶体熔化时温度持续上升

冰是晶体，熔点是 0℃。冰在0℃时开始熔化，熔化过程中温度保持0℃不变，直到全部熔化成水后温度才继续上升。

晶体熔化条件：① 达到熔点 ② 继续吸热

✦ ✦ ✦

4. 冰桥渡河

继续前进了一段距离，一条湍急的河流拦住了去路。

河水翻滚着，冒着热气——水温至少七八十度，不可能直接趟过去。河上也没有桥。

"我们怎么过去？"凌音犯愁了。

林小宇环顾四周，目光落在河边的一些岩壁上——那里的温度极低，岩壁上挂着厚厚的冰凌。

"有了。"他说，"我们可以造一座冰桥。"

"怎么造？"凌音不解。

林小宇走到冰凌旁，集中注意力。他想起了物理课上学的：凝固——水从液态变成固态的过程，需要放热。如果能快速从水中吸走热量，水就会凝固成冰。

他伸出手，掌心对准河面。在"观察者之眼"的视野中，他看到了河水中的热能——那些热能粒子在疯狂运动。他用意念引导河水中的热流向四周散发（实际上是周围寒冷的空气吸走了热量），河水表面的温度急剧下降——

咔咔咔——

河面上出现了一层薄冰。薄冰迅速加厚，几分钟之内，一座宽约一米的冰桥横跨了河面。

林小宇满头大汗。控制温度比控制声音要费力得多。但他做到了——他利用凝固的原理，让河水放热变成了冰。

"不可思议……"凌音轻声说，"你居然能主动控制物态变化。"

"是熔化和凝固的原理。"林小宇擦了擦汗，"物质从液态变成固态——凝固——需要放热。我让河水快速散热，它就结冰了。"

两人小心翼翼地走过冰桥。当最后一个人踏上对岸时，身后的冰桥因为周围的热量而开始熔化，很快消失在河水中。

"真险。"凌音说。

林小宇却若有所思。他开始意识到，熔化、凝固、汽化、液化——这些以前在课本上需要死记硬背的词语，在这个世界里是实实在在的力量。

⚡ 法则笔记：六种物态变化总览

物质有三种状态：固态、液态、气态。它们之间可以相互转化——这就是物态变化。

❄️ 固态
（冰）
→ 熔化(吸热) →
💧 液态
（水）
→ 汽化(吸热) →
♨️ 气态
（水蒸气）
❄️ 固态
（冰）
← 凝固(放热) ←
💧 液态
（水）
← 液化(放热) ←
♨️ 气态
（水蒸气）
❄️ 固态
↔ 升华/凝华 ↔
♨️ 气态

吸热过程（外界加热）：熔化、汽化、升华

放热过程（向外界放热）：凝固、液化、凝华

记忆口诀："熔汽升吸热，凝液固化热"

5. 山谷中的营地

夜幕降临，炎寒谷的气温骤降。林小宇和凌音在谷中的一块高地上扎营，周围用岩石围成一圈，防止冷风进入。

凌音生了一堆火。火焰在夜色中跳动，给两人带来了温暖。

"你是怎么学会控制温度的？"凌音问。

林小宇想了想。"在我们那个世界，这些知识写在课本里。但我从来没认真学过……直到今天。"他看着自己的手掌，那里有淡淡的法则之线在流动。"现在我才真正理解它们是什么意思。"

"你觉醒了法则之力，而且进步飞快。"凌音拨了拨火堆，"也许……你真的能收集所有碎片，修复元界的法则。"

"那你呢？"林小宇问，"你为什么在漂移之原流浪？你的家在哪？"

凌音沉默了一会儿。"我不记得了。"

"不记得了？"

"我只记得——我醒来时就在漂移之原，身上只有这根笛子和温度计。我脑海中有一些关于法则的知识，但我不知道是从哪里来的。"她抬起头，看着紫色的夜空，"也许等我帮你收集所有碎片，我就能找到答案。"

林小宇看着她，忽然觉得这个少女身上藏着很多秘密。但他没有追问——每个人都有不想说的事。

他躺下来，看着头顶的星空。两颗月亮挂在天空，一红一蓝，交相辉映。

"明天我们继续深入。"他说，"我已经感觉到了——温衡之石就在山谷的最深处。"

🧪 动手探索：冰的熔化实验

🧊 观察冰的熔化

在家里用冰块和温度计，亲眼看到熔化时温度不变的现象！

  1. 从冰箱取出几块冰，放入杯子中
  2. 用温度计测量冰块的温度（应该是0℃以下）
  3. 将杯子放在室温下，每分钟记录一次温度
  4. 你会发现：温度先上升，到0℃时停下来——冰正在熔化！等到冰全部熔化后，温度才会继续上升
🧊 自制冰激凌（实验小奖励）
  1. 在小袋子里装入牛奶、糖、几滴香草精，封好口
  2. 在一个大袋子里装入冰块和盐（盐可以降低冰的熔点，让温度更低！）
  3. 把小袋子放进大袋子，封好口，用力摇晃10分钟
  4. 打开看看——牛奶凝固成了冰激凌！这就是凝固和熔点降低的应用！

6. 山谷深处的脚步声

夜深了。火堆发出噼啪的声响。

林小宇正要入睡，忽然听到远处传来一阵沉重的脚步声——轰、轰、轰——像是某种巨兽在行走。

凌音也醒了。她握紧笛子，警觉地看着黑暗深处。

"是守护者。"她低声说，"炎寒谷的守护者——霜炎巨兽。"

脚步声越来越近。地面在震动。黑暗中，林小宇隐约看到一对巨大的眼睛——一只眼睛冒着蓝光，像冰一样寒冷；另一只眼睛冒着红光，像火一样炽热。

"它来了。"凌音说。

林小宇站起身，掌心凝聚起法则之力。他知道——真正的试炼，才刚刚开始。
