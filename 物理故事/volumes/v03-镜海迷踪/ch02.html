第二章

幻境深处

📖 物理知识点：光的折射 · 凸透镜与凹透镜 · 透镜成像

✦ ✦ ✦

1. 光的预判

镜海蛇妖操纵着数十面巨镜，光线从四面八方射来。林小宇激活观察者之眼，全世界的速度仿佛都慢了下来。

他"看到"了——每一道入射光的路径，每一次反射后的新方向。在脑海中，他自动计算出蛇妖下一次攻击的落点。

"左边三秒后！"他拉着凌音向左跳开。一道光柱正好击中他们刚才的位置。

"右边五秒后，高度两米！"凌音蹲下，光柱从头顶擦过。

蛇妖的镜像在每一面镜子里做出不同的动作，让它看起来像是有数十个分身同时攻击。但林小宇发现了一个规律——无论它在镜子里有多少个分身，真正的本体只有一个。蛇妖之所以能控制所有镜像，是因为它利用光线在不同镜面之间快速移动视线。

"它的本体在那里！"林小宇指向一面巨镜的后面——那个最不起眼的角落。

但问题是怎么攻击到它？蛇妖被镜子保护着，所有攻击都会被镜面反射回来。

林小宇快速思考——镜子反射光，但并不是所有东西都反射光。如果光从一种透明介质进入另一种透明介质，比如从空气进入水中，它不会全部反射，而是会发生折射——传播方向发生偏折。

"镜海的海水！"他灵光一闪，"水面虽然像镜子一样反射，但也有一部分光会折射进入水下！如果能让蛇妖沉入水中，镜子战术就不管用了！"

✦ ✦ ✦

2. 光的折射

凌音明白了他的计划。她吹响笛子，音波在水面下引发共振——

轰——！

蛇妖脚下的镜面碎裂，它连同镜子一起掉入水中！

林小宇趁机跳入水中——当然，他不会游泳，但凌音用音波制造了一个气泡。

在水下，林小宇看到了奇妙的现象：光线从空气射入水中时，方向发生了偏折。原本直的筷子看起来像被折断了——这是光的折射。

蛇妖在水下失去了镜子战术的优势，开始用身体直接攻击。林小宇利用折射现象预判光线在水下的传播路径——光线从水射入空气中时，折射角大于入射角。

他根据这个原理调整了自己的观察角度。蛇妖在水下的实际位置和眼睛看到的略有偏差——他需要计算出这个偏差才能准确攻击。

⚡ 法则笔记：光的折射

光从一种介质斜射入另一种介质时，传播方向发生偏折——这就是光的折射。

折射规律：

  • 光从空气斜射入水/玻璃中 → 折射光线靠近法线（折射角 < 入射角）
  • 光从水/玻璃斜射入空气中 → 折射光线远离法线（折射角 > 入射角）
  • 垂直入射时 → 方向不变

生活中的折射现象：

  • 筷子在水中看起来"折断"了
  • 池水看起来比实际浅
  • 从岸上看水中的鱼，它的位置看起来比实际浅

光从空气→水中：折射角 < 入射角

✦ ✦ ✦

3. 凸透镜的威力

蛇妖被激怒了。它从水中腾跃而出，带着无数镜片，在空中形成了一面巨大的曲面镜——不对，那不是平面镜，而是凸面镜！

凸面镜将光线向四周发散，形成了一大片眩光区域。林小宇和凌音被强光照射，暂时睁不开眼睛。

"它在利用曲面改变光的会聚方式！"林小宇捂着眼睛喊道。

他强迫自己冷静下来。镜子可以是平面的，也可以是曲面的。凸面镜发散光线……那如果反过来呢？

他环顾四周，看到了一块透明的、形状像放大镜的石头——那是一个天然的凸透镜！

物理课上的知识涌上心头：凸透镜对光有会聚作用，平行光经过凸透镜后会会聚在焦点上。而凹透镜对光有发散作用。

他捡起那块凸透镜形状的石头，举到太阳光下。光线经过凸透镜后会聚到一个点上——焦点。焦点的温度急剧升高——

嗤——！

焦点处的镜面开始冒烟、熔化！

"太棒了！"林小宇兴奋地大喊。他用凸透镜汇聚阳光，像用放大镜点火一样，将焦点对准蛇妖周围的镜面。一面面镜子在高温下熔化变形，失去了反射能力。

⚡ 法则笔记：凸透镜与凹透镜

透镜：由透明材料制成的光学元件，两个面都是球面的一部分。

凸透镜（中间厚、边缘薄）：

  • 对光有会聚作用（汇聚透镜）
  • 平行光 → 会聚于焦点（F）
  • 焦点到透镜中心的距离 → 焦距（f）

凹透镜（中间薄、边缘厚）：

  • 对光有发散作用（发散透镜）
  • 平行光 → 被发散，发散光线的反向延长线交于焦点

📝 记忆："凸聚会聚，凹发散散"

应用：凸透镜——放大镜、照相机、投影仪、远视眼镜
凹透镜——近视眼镜、门上的猫眼

✦ ✦ ✦

4. 透镜成像

蛇妖的镜面护甲被凸透镜破解了大半，但它仍然不肯屈服。它用最后几面完好的镜子制造出一个巨大的幻象——

一面无比巨大的虚幻蛇影从海面上升起，张开了血盆大口。

"这是它的最后一招了！"凌音喊道，"它把光线汇聚成一个巨大的虚像来吓唬我们！"

林小宇盯着那个巨大的蛇影。在观察者之眼的视角中，他清楚地看到——那只是光的把戏。蛇妖通过特定排列的镜子和透镜，把一个小模型放大成了一个巨型幻象。

这就像凸透镜成像的规律——当物体放在凸透镜的焦点以内时，会在物体的同一侧形成一个放大的虚像。

"它用的是放大镜的原理。"林小宇说，"它自己很小，但通过凸透镜形成一个巨大的虚像来吓退敌人。"

"那怎么打败它？"凌音问。

林小宇回想透镜成像的规律——凸透镜成像有五种情况：

  • 物体在二倍焦距以外 → 倒立缩小的实像（照相机）
  • 物体在二倍焦距上 → 倒立等大的实像
  • 物体在一倍焦距和二倍焦距之间 → 倒立放大的实像（投影仪）
  • 物体在一倍焦距上 → 不成像
  • 物体在一倍焦距以内 → 正立放大的虚像（放大镜）

蛇妖利用的就是最后一种——物体在一倍焦距以内形成放大虚像。如果他能移动到蛇妖的透镜的焦点以外，蛇妖的幻象效果就会消失！

"凌音，帮我掩护！我要冲到它的透镜后面去！"

凌音吹响笛子，音波在空气中形成屏障，挡住了蛇妖的散射光线。林小宇借着掩护，绕到蛇妖的背后——果然，那里有一个小巧的凸透镜装置。蛇妖躲在透镜后面，看起来巨大无比，实际上本体很小。

林小宇一把抓住了那个透镜！

失去了透镜的放大作用，蛇妖的幻象瞬间消失。它的真身暴露了出来——只有手臂粗细的一条小蛇，鳞片闪着微光。

"你……你破解了我的光之幻术……"蛇妖虚弱地说。

"光不是用来欺骗的。"林小宇说，"光是用来照亮世界的。"

蛇妖的身体化作无数光点消散。那些光点在空中旋转汇聚，最终凝结成一块透明的石头——光芒四射，像一颗晶莹的钻石。

第三块法则碎片——光耀之石。

⚡ 法则笔记：凸透镜成像规律

凸透镜成像的五种情况（u=物距，v=像距）：

物距 u像的特点像距 v应用
u > 2f倒立缩小的实像f < v < 2f照相机
u = 2f倒立等大的实像v = 2f测焦距
f < u < 2f倒立放大的实像v > 2f投影仪
u = f不成像—平行光源
u < f正立放大的虚像|v| > u放大镜

📝 记忆口诀："一倍焦距分虚实，二倍焦距分大小"

✦ ✦ ✦

5. 光耀之石

林小宇捡起光耀之石。石头入手温热，他能感受到光的法则之力在体内流动——从光的直线传播到反射定律，从折射规律到透镜成像，所有的光学知识在这一刻融会贯通。

他的"法则之力"再次升级了。他现在不仅能看到光的传播路径，还能在短时间内偏折光线的方向——虽然消耗很大，但已经能做到初步的光线操控。

"三块了。"凌音微笑着说，"静止之石、温衡之石、光耀之石……你已经收集了三块碎片。"

林小宇举起光耀之石。三块碎片在他的掌心上方悬浮，开始缓慢旋转。一幅更完整的地图在脑海中展开——第四块碎片在遥远的西方，那里标注着……

"重铸之城。"林小宇念出了这个名字。

"我听说过那里。"凌音说，"一座正在崩溃的城市——金属浮在空中，木头沉入地下。那里的人违背了密度和质量的基本法则。"

林小宇看着远方。他已经掌握了运动、物态变化和光的法则。前方还有更多挑战——密度、力、压强、浮力、机械能、电、磁……

但他已经不再害怕了。每掌握一个法则，他就离回家的路更近一步。

他把光耀之石握在手中，感受着光的温度。"走吧。下一站——重铸之城。"
