第2章：浮生之战

知识点：浮力 · 阿基米德原理 · 浮沉条件

修理好调培塔后，林小宇和凌音在城中休整了一夜。第二天一早，老雷带来了一个令人不安的消息——深渊底部的能量波动越来越强，整座城市的地基都在微微震动。

"深渊巨兽——如果它真的苏醒了，我们需要启动'浮力锚'。"老雷说。

"浮力锚？"林小宇好奇地问。

"那是古代建造者留下的终极防御系统。它的核心是一排巨大的浮力舱——当城市某处受到巨大压力或下沉威胁时，浮力舱可以产生足够大的浮力，把整座城市托住。"老雷叹了口气，"但是浮力锚也已经损坏了。我们谁都不懂怎么修。"

"带我去看看。"林小宇说。

⚡ 法则笔记：浮力

浮力是浸在液体（或气体）中的物体受到的向上的托力。

浮力的方向：竖直向上。

浮力产生的原因：液体对物体上下表面的压力差。

阿基米德原理：

F浮 = G排 = ρ液 g V排

浸在液体中的物体所受的浮力，等于它排开的液体所受的重力。

这个故事传说中，阿基米德在浴缸里发现这个原理时，激动得光着身子跑上大街大喊"尤里卡！"（我找到了！）

浮力锚位于城市底部的一座穹顶大厅中。一排巨大的金属舱体悬浮在一个深水池中——或者说，原本应该悬浮着。现在大多数舱体都沉在了池底，只有少数几个还漂在水面上。

"看这个——"凌音走到一个控制台前，上面显示着一组读数。"每个浮力舱的排开水量都变了。原来设计的时候，每个舱体排开的水的体积正好能产生足够的上浮力，但现在——"

林小宇走近一个沉底的舱体，把手放在上面。他闭上眼睛，脑海中出现了这个舱体的"受力分析"：

舱体受到向下的重力（G）和向上的浮力（F浮）。当 G > F浮 时下沉，G < F浮 时上浮，G = F浮 时悬浮。

"这个舱体进水了。"林小宇睁开眼，"它内部被灌入了水，使得整体的重量变大，重力超过了浮力，所以就沉下去了。"

"那只要把水排出来就行？"凌音问。

"对！但我们需要找到每个舱体的排水阀，而且要确保排完水后浮力足够大。"林小宇走向下一个舱体，"同时我们还要调整一些舱体的体积——V排 越大，浮力越大。"

⚡ 法则笔记：浮沉条件

浸在液体中的物体的浮沉取决于物体所受浮力和重力的大小关系：

🔴 下沉：F浮 < G → ρ液 < ρ物（如铁块在水中下沉）

🟢 上浮：F浮 > G → ρ液 > ρ物（如木块在水中上浮）

🟡 悬浮：F浮 = G → ρ液 = ρ物（如悬浮在水中的鱼）

🔵 漂浮：F浮 = G，但 V排 < V物 → ρ液 > ρ物（如船浮在水面）

应用举例：

• 潜水艇：通过改变自身重力来实现浮沉（向水舱注水/排水）

• 热气球：通过改变空气密度（加热气体）来改变浮力

• 轮船：用"空心"的方法增大 V排，从而获得更大的浮力

林小宇和凌音花了大半天时间，挨个检查了十二个浮力舱。他们排出了进水的舱体，修复了几个破损的阀门，还调整了一个舱体的大小以增大浮力。

当最后一个舱体被修复完毕，浮力锚系统重新启动时，整座城市发出了轻微的震动——然后稳定了下来。深渊底部的能量波动似乎也被压制住了。

"干得好。"一个陌生的声音从身后传来。

林小宇和凌音猛地转身。一个全身笼罩在暗紫色斗篷中的人影站在穹顶入口处——暗影使徒。

"就是你把调培塔破坏的？"林小宇质问道。

"破坏？"暗影使徒轻笑了一声，"我只是加速了一个必然的过程。深渊之底的能量不是你想象的那样简单——不过，你已经证明了自己的实力。"

他伸出手，掌心出现一个旋转的暗紫色光球："我在齿轮森林等你。如果你想了解这个世界的真相——就来吧。"

说完，暗影使徒化作一道暗影消失了。

"他到底想干什么？"凌音皱起眉头。

"不管他想干什么——"林小宇握紧拳头，"我们必须去齿轮森林。他说得对，这个世界的真相，我必须搞清楚。"

🧪 动手探索：浮力的奥秘

🥚 实验1：鸡蛋的浮沉

目标：通过改变液体密度来改变浮力，让鸡蛋浮起来。

材料：生鸡蛋、透明杯子（或玻璃瓶）、水、盐、勺子、筷子。

  1. 在杯子里装大半杯水，轻轻放入鸡蛋——鸡蛋沉入水底
  2. 向水中逐渐加盐，用筷子搅拌溶解
  3. 加到一定程度时，鸡蛋会慢慢浮起来——最终悬浮在水中
  4. 继续加盐，鸡蛋会浮到水面上

原理：加盐增大了水的密度（ρ液增大），使 F浮 = ρ液gV排 增大。当 F浮 > G 时，鸡蛋上浮。

🛳️ 实验2：橡皮泥船

目标：理解"空心法"增大浮力的原理。

材料：一块橡皮泥（或黏土）、盆子、水、几枚硬币。

  1. 把橡皮泥揉成球状，放入水中——它会沉入水底
  2. 把橡皮泥捏成碗状或船形（尽量捏薄，边缘捏高）
  3. 再次放入水中——它会浮在水面上
  4. 往"船"里慢慢放硬币，看看最多能放几枚硬币才沉没

原理：同样重量的橡皮泥，做成船形后排开的水更多（V排更大），产生的浮力更大。这就是为什么钢铁造的轮船能浮在水面上！
