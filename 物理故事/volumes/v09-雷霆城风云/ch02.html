第2章：暗影总部

知识点：欧姆定律 · 电功率 · 焦耳定律

通过城门后，林小宇和凌音进入了雷霆城的内城区。这里的建筑更加高大精密，空气中充满了电流的嗡嗡声。

城市中央矗立着一座黑色的高塔——暗影使徒的总部。塔的周围环绕着高压电网，发出噼啪的电弧声。

"我们需要关闭电网才能进入。"凌音观察着电网的供电系统，"那有一个控制箱。"

他们走近控制箱。箱子上显示着电压和电流的读数，旁边有一个需要调节的旋钮。

林小宇看着读数："电压U=220V，电阻R=……不知道。但我们可以用欧姆定律来计算电流。"

⚡ 法则笔记：欧姆定律

欧姆定律：导体中的电流，跟导体两端的电压成正比，跟导体的电阻成反比。

I = U / R

I 是电流（A），U 是电压（V），R 是电阻（Ω）。

电阻（R）：导体对电流的阻碍作用。

• 单位：欧姆（Ω），千欧（kΩ），兆欧（MΩ）

• 电阻是导体本身的属性，与电压和电流无关

• 影响电阻大小的因素：材料、长度、横截面积、温度

• 同种材料：长度越长电阻越大，横截面积越大电阻越小

串联电路（等效电阻）：R = R₁ + R₂ + …

并联电路（等效电阻）：1/R = 1/R₁ + 1/R₂ + …

凌音指着控制箱内的一组电阻说："这是一个串联电路——两个电阻串联后接入220V电路。总电阻等于两个电阻之和。如果我们改变接入电路的电阻值，就能控制电流的大小。"

林小宇调整了旋钮，改变了电路中的电阻，电流随之变化。控制箱上的红灯变成了绿灯——电网的功率降低了。

"现在我们进去吧。"凌音说。

他们走进黑塔。塔内是暗影使徒的实验室，到处是精密的电子设备和仪器。墙上挂满了电路图和能量流动图。

在实验室的中央，林小宇看到了一个巨大的机器——一个散发着暗紫色光芒的能量核心。

"你终于来了。"暗影使徒的声音从背后响起。

林小宇和凌音转过身。暗影使徒摘下了兜帽——露出一张年轻的脸，看起来很疲惫。

"你……"林小宇惊讶地说不出话来。

"我叫暗影——或者你可以叫我……曾经的元界守护者。"他苦笑着说，"你们一定有很多问题。"

"为什么你一直在暗中引导我们？"凌音问道。

"因为我在修复一个错误。"暗影走向能量核心，"元界的能量系统正在崩溃——所有物理法则都在逐渐失效。如果不在完全崩坏之前修复核心，整个元界都会毁灭。而我——"他看向林小宇，"需要你的帮助。"

⚡ 法则笔记：电功率与焦耳定律

电功率（P）：电流做功的快慢。

P = UI

P 是电功率（瓦特 W），U 是电压（V），I 是电流（A）。

还可以推导出：P = I²R = U²/R

额定电压与额定功率：

用电器上标明的电压和功率，如"220V 100W"。

焦耳定律：电流通过导体产生的热量。

Q = I²Rt

Q 是热量（J），I 是电流（A），R 是电阻（Ω），t 是时间（s）。

这就是电热器（电热水壶、电饭锅）的原理：利用电流的热效应。

"元界为什么会崩坏？"林小宇问。

"因为有人——一个来自你们世界的科学家——发现了元界的存在，并试图抽取元界的能量。他破坏了法则核心。"暗影看着能量核心，"我的力量不足以修复它，但你可以。你是唯一能看到物理法则的人。"

"我需要做什么？"林小宇问。

暗影指向能量核心周围的一组装置："这些装置控制着元界的物理常数。它们需要按照正确的电功率运行——太大或太小都会导致能量不稳定。我必须精确计算出每个装置需要的功率。"

林小宇走上前。他看到每个装置上都标有电压和电阻。他默默地计算着：

P = U²/R —— 知道了电压和电阻，就能算出功率。知道了功率，就能判断装置是否需要调整。

他花了几个小时，一个接一个地调整了所有装置。当最后一个装置调整完毕时，能量核心发出了稳定而明亮的光芒。

"完成了。"林小宇松了一口气。

暗影点了点头："还有最后一件事——去磁之秘境。那里是元界的能量之源，也是你回家的唯一通道。"

🧪 动手探索：电学实验

💡 实验1：灯泡亮度的秘密

目标：理解电功率与灯泡亮度的关系。

材料：两个额定电压相同但额定功率不同的灯泡（如"2.5V 0.3A"和"2.5V 0.5A"）、两节电池、导线、开关。

  1. 分别将两个灯泡接入相同的电路中
  2. 闭合开关，观察两个灯泡的亮度差别
  3. 哪个灯泡更亮？——额定功率更大的灯泡更亮
  4. 如果条件允许，将两个灯泡串联接入电路，观察亮度变化

原理：灯泡的亮度由实际功率决定。P = UI，电流越大或电压越大，功率越大，灯泡越亮。

🌡️ 实验2：电流的热效应

目标：观察电流通过导体时产生热量的现象。

材料：一段细铁丝（或电阻丝）、两节电池、导线、一小段火柴梗或牙签。

  1. 把细铁丝的两端连接到电池的正负极上（注意：铁丝会发热！）
  2. 用手小心地感受铁丝的"附近"温度（不要直接触摸，以免烫伤）
  3. 在铁丝上放一小段牙签或火柴梗，观察它是否会冒烟
  4. 尝试用更细的铁丝或更多节电池，观察发热的变化

原理：Q = I²Rt。电流越大、电阻越大、时间越长，产生的热量越多。这就是电热毯、电热水壶的原理。
