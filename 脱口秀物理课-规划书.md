# 🎤 脱口秀物理课 · 整体规划书

> 学物理不该那么枯燥——用脱口秀的方式，让学生在欢声笑语中掌握物理知识。
> 结合已有项目「物理解码者：异世法则」的故事素材，打造全新教学体验。

---

## 一、项目基本面（已确认）

| 维度 | 决策 |
|------|------|
| **人设** | 外表沉稳、内心热情的物理老师（冷面笑匠型） |
| **发布形式** | 公众号逐字稿（画面感强，读者能"看见"场景） |
| **品牌关系** | 独立品牌，与「物理解码者：异世法则」互补联动 |
| **更新节奏** | 每周 2 期 |
| **创作者身份** | 物理老师 + 内容创作者 |

### 人设画像

```
外表：沉稳、淡定、波澜不惊
着装：简约（白衬衫/纯色T恤）、干净利落
语气：不紧不慢、娓娓道来
内核：对物理有滚烫的热爱，讲到精彩处眼神发光
风格：不刻意搞笑，用对物理的热情和精准的洞察自然带出笑点
类比：像一位冷静的科学家突然被问到自己的研究领域——瞬间变身激情安利达人
```

## 二、核心理念

| 维度 | 描述 |
|------|------|
| **核心口号** | "笑着笑着就学会了" |
| **目标人群** | 初中生（八年级~九年级），尤其物理偏科、觉得物理枯燥的学生 |
| **差异化** | 不堆砌公式，不讲大道理 —— 用段子解构知识，用生活场景拉近距离，用对物理的热情感染学生 |

**一句话**：把物理老师的教案改写成脱口秀稿子，把教室变成开放麦现场。

## 三、内容体系规划

### 3.1 知识点覆盖（对标人教版初中物理）

| 专场 # | 专场名 | 知识点 | 段子方向 |
|--------|--------|--------|---------|
| 0 | **开场秀：物理是个啥？** | 物理是什么、为什么要学物理 | 吐槽学生眼中"最没用的学科" |
| 1 | **卷王的世界** | 机械运动·参照物·速度 | 为什么你坐在教室里不动也算"运动" |
| 2 | **闭嘴，听我说！** | 声音的产生与传播·回声·音调响度 | 班主任的"超声波"攻击 |
| 3 | **你热了吗？** | 温度·物态变化·熔化凝固 | 冰棍融化速度 vs 暑假作业进度 |
| 4 | **冰火两重天** | 汽化液化·升华凝华 | 洗澡时镜子为什么起雾 |
| 5 | **我看透了这个世界** | 光的直线传播·反射 | 为什么自拍要用美颜——镜子里的是你吗？ |
| 6 | **折射人生** | 光的折射·凸透镜成像 | 筷子插水里弯了，像极了你的理综成绩 |
| 7 | **重量级选手** | 质量与密度 | 同款奶茶大杯和中杯到底差在哪 |
| 8 | **你别推我** | 力·重力·摩擦力·惯性 | 为什么急刹车时你飞出去 |
| 9 | **压力山大** | 压强 · 液体压强 · 大气压 | 学霸坐你旁边为什么你没感到压强 |
| 10 | **浮起来吧少年** | 浮力 · 阿基米德原理 | 你在浴缸里泡澡和"啊！我发现了！" |
| 11 | **给我一个支点** | 简单机械·杠杆·滑轮 | 为什么你爸撬不开你的房门 |

| 15 | **磁力吸引法则** | 电与磁·电磁感应 | 为什么有些人对你有"引力"——并不是 |

### 3.2 内容结构（每个专场的标准结构）

```
🎬 **开场热场**（1-2 min）
  → 沉稳切入——"今天我们来聊一个跟你们每个人都有关的事……"
  → 与上期做 call-back
  → 语调平稳，但眼神开始发光

🔥 **段子时间**（5-8 min）
  → 3-4 个生活段子逐步引出核心知识点
  → 外观冷静拆解，内里热烈翻包袱
  → 每个段子末尾翻出物理原理

📖 **法则小课堂**（3-5 min）
  → 掏出「物理解码者」中的相关故事情节作为深度案例
  → 将异世界场景与学生生活做类比
  → 公式做成梗——"v=s/t 不是魔法，是魔法数学"

💡 **互动挑战**（2-3 min）
  → 给一个生活现象，让读者猜物理原理
  → 预埋"梗选项"

🎯 **本期金句 + 下期预告**（1 min）
  → 一句话总结，要有记忆点和传播力
  → 预告下期，埋钩子
```

## 四、微信公众号逐字稿指南

### 4.1 排版风格

| 元素 | 风格 |
|------|------|
| 排版原则 | 留白多、段落短（每段不超过 4 行）、呼吸感强 |
| 口语化 | 完全口语写作，像老师在讲台上说话 |
| 画面感 | 每段话让读者能"看到"场景——补充视觉描写 |
| 互动感 | 用"你们想想……"、"来，我问你们一个扎心的……"拉近距离 |
| emoji | 适度使用，每节一个核心emoji引导 |

### 4.2 逐字稿 vs 普通文章的区别

```
❌ 普通科普文章：
"参照物是判断物体运动状态的参考标准。选择不同的参照物，
对同一个物体的运动描述可能不同。"

✅ 脱口秀逐字稿：
"来，我问你们一个问题——你现在是静止的，还是在运动？
（停顿）
大部分人会说：静止的。我坐在椅子上，一动没动。
好，那我告诉你——你错了。（停顿）你正以每秒约30公里的速度在太空中飞驰。
（停顿，让信息消化）
你看，这就是参照物的魔力——你觉得你没动，是因为你选了地球当参照物。
但如果你选了太阳当参照物……你就是一个每秒飞30公里的超人。"
```

### 4.3 画面感增强技巧

- 场景描写："想象一下，你坐在教室里，窗户半开，风吹进来……"
- 人物动作："（停顿，推了推眼镜）你们猜接下来发生了什么？"
- 表情暗示:"（眼神突然亮起来）这个现象，物理可以解释"
- 节奏标记："（停顿）""（笑）""（压低声音）"
