EP 03

你热了吗？

❄️ 冷面物理 · 笑着笑着就学会了

📖 温度·温度计·熔化·凝固·晶体与非晶体
✦ ✦ ✦
🎬 开场

（停顿，推了推眼镜）

今天我们来聊一个跟你们每个人都感同身受的话题——

热。

不是"热"本身，是"你热了吗"的那个热。

（语速平稳）

你有没有发现一个很奇怪的现象——

夏天 30 度，你热得像个冰淇淋一样融化。

冬天 10 度，你冷得像个冰棍一样发抖。

但这中间的 20 度温差，跟物理实验里那些动不动就零下 200 度、零上 1000 度比起来——简直温柔得像一杯温水。

那问题来了：温度到底是什么？

（停顿，看着大家）

你可能会说："温度嘛，就是冷热程度呗。"

对，也不全对。

物理上，温度的本质是——分子运动的剧烈程度。

（语速放慢，眼神开始发亮）

你感觉"热"，不是因为天气跟你过不去。

是因为空气分子在疯狂地撞击你的皮肤，每秒撞几亿次。

你感觉"冷"，也不是天气针对你——

是因为分子运动慢了，给你的撞击少了。

所以下次有人说你"热情"——

你的分子运动很剧烈，说明你体内的分子正在高速运转。

（淡淡地）

反之，有人说你"冷淡"——

你的分子运动比较平缓。

这是科学，不是态度问题。

✦ ✦ ✦
🔥 段子时间
  • 温度计——我"体感"不算数

（拿出一个温度计的动作）

你们都用过体温计吧？

小时候发烧，你妈把体温计夹在你腋下。

五分钟后拿出来一看——38.5 度。

你妈说："发烧了，吃药。"

你说："我觉得还好……"

你妈说："体温计说的不算你说的。"

（停顿）

这就是物理的第一课——客观测量。

你觉得热不热，那是"体感"。

温度计上的数字，那是"温度"。

体感会骗你。

温度计不会。

你去洗桑拿，80 度的空气你还能待一会儿。

但如果是 80 度的水——你进去就熟了。

为什么？

因为空气和水的比热容不一样，传递热量的速度和能力也不一样。

但温度计上的数字都是 80。

温度没骗你——是你自己的感觉体系太脆弱了。

（微微摇头）

所以物理教会我们的第一件事就是：

别信感觉，信数据。

就像你觉得自己复习了三个小时——

看看计时器吧，可能只有 40 分钟。

  • 温度的变化——从冰棍到"水棍"

（不紧不慢地）

夏天你们最爱吃什么？

冰棍。

但你们有没有发现一个奇怪的现象——

你从冰箱里拿出一根冰棍，在室温下放着。

它开始融化。

滴答，滴答，水一滴一滴掉下来。

（停顿）

但你知道吗——冰棍在融化成水的过程中，温度一直是 0 度。

对。你没听错。

冰在融化的时候，虽然你看到它在"变水"，但它的温度不变。

（眼神微微发亮）

这就是物理上一个非常重要的概念——熔化和凝固。

熔化：固体变成液体的过程。

凝固：液体变成固体的过程。

关键在于——在熔化/凝固的过程中，温度不变。

冰变成水，温度保持在 0 度，直到所有冰都融化成水，温度才会继续上升。

水结成冰，温度也保持在 0 度，直到所有水都结成冰，温度才会继续下降。

（停顿）

这意味着什么？

意味着你跟物理学家说："这块冰正在变水，但它还是 0 度。"

物理学家说："正常。"

你说："但这不合理啊——它在变，但温度不变？"

物理学家说："它就是不变。"

（笑）

物理有时候就是这么"不讲道理"，但它是真的。

所以冰棍滴下来的不是"融化了的冰"——

那是 0 度的水。

  • 晶体和非晶体——有原则的物质和没原则的物质

（推了推眼镜）

既然讲到熔化和凝固，我必须跟你们说一个很酷的分类——

有些物质，有"原则"。

有些物质，没有"原则"。

有原则的：晶体。

比如冰、食盐、钻石、金属。

它们有固定的熔点和凝固点。

冰的熔点是 0 度——不到 0 度不化，到了 0 度才开始化，而且在化完之前温度不涨。

就像一个有原则的人——

"我说 0 度就是 0 度，少一度不化，多一度才开始。没商量。"

（停顿）

没原则的：非晶体。

比如玻璃、橡胶、塑料、沥青。

它们没有固定的熔点。

从固体变软、变黏、变稀——是一个渐变的过程，没有一个精确的温度点。

就像你爸说 "马上到"——

"马上"可以是 5 分钟，也可以是 50 分钟。

没有一个精确的时间点。

（笑）

所以下次你上课学"晶体有固定的熔点"——

你就记住：冰是那个说"0 度就 0 度，少一度都不行"的倔强者。

而玻璃是那个说"呃……我大概……可能……快化了？"的犹豫者。

  • 一个关于"热"的人生哲理

（语速放缓）

你们不觉得物理很有趣吗？

它研究的是最基础的东西——冷、热、固体、液体、气体。

但它教会我们的道理，却远远超过了物理本身。

比如：

晶体熔化需要吸收热量，但温度不变。

这意味着什么？

意味着你在成长的某些阶段——你看起来很努力，你的"温度"没有变化。

但你在"吸收热量"。

你在积累。

你在为下一步的变化储备能量。

等到所有该融化的都融化了——

温度才会开始上升。

（停顿）

所以，如果你现在觉得自己努力了但看不到进步——

你不是在浪费时间。

你是在"熔化"。

你的温度没变，但你在吸收热量。

等到了那个临界点——你会发现自己已经完全不一样了。

📖 法则小课堂

收了。正经 60 秒。

今天三个核心知识点：

  • 温度：表示物体冷热程度的物理量。本质是分子运动的剧烈程度。

口诀："热=分子嗨，冷=分子蔫。"

  • 熔化与凝固：熔化（固→液）吸热，凝固（液→固）放热。熔化/凝固过程中温度不变。

口诀："熔化吸收热，凝固释放热，温度不变是铁律。"

  • 晶体与非晶体：
  • 晶体：有固定熔点（冰、食盐、金属）
  • 非晶体：没有固定熔点（玻璃、橡胶、塑料）

口诀："晶体有原则，非晶体没谱。"

说到这个，「物理解码者」V02C01 冰与火之门里——

林小宇进入炎寒谷，一个温度极不稳定的地方。

水在冰、水、蒸汽之间胡乱跳变，因为他进入的是一个"物态变化失控"的领域。

他能活下来，就是因为他在课堂上学过——

物质的状态不是魔法，是温度决定了它们是谁。

想知道他怎么用物态变化知识破解机关的吗？去看异世法则吧。

💡 互动挑战

来，考考你——

（停顿）

一杯 0 度的冰水混合物，放在 0 度的房间里。

问：冰会融化吗？

A. 会，因为冰本来就是会融化的

B. 不会，因为冰水混合物和房间温度相同，没有热传递

C. 会融化一半然后停止

（停顿）

答案是 B. 不会。

冰融化需要吸收热量。但房间也是 0 度——没有温度差，就没有热传递。冰吸收不到热量，就老老实实地当冰。

所以物理告诉你：要融化别人，首先要让自己有温度差。

（微微点头）
