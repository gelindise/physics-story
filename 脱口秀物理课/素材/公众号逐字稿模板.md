# 📝 冷面物理 · 公众号逐字稿模板

> 使用说明：每期按此模板填充，保持风格统一。
> 品牌暂用名「冷面物理」，正式发布前可替换。

---

## 排版规范

| 元素 | 规则 |
|------|------|
| 正文字号 | 15-16px（公众号默认） |
| 段落长度 | 每段不超过 4 行，留白充足 |
| 括号动作 | `（停顿）` `（笑）` `（推眼镜）` 模拟现场节奏 |
| 引用强调 | **粗体** 标出核心概念 |
| 画面描写 | 关键场景前加一两句环境描写 |
| emoji | 每节开头 1 个，不过度使用 |
| 分割线 | `—— · —— · ——` 分隔大段 |

## 逐字稿示例（片段）

```
❌ 普通写法：
声音的传播需要介质，真空不能传声。

✅ 冷面物理风格：

（停顿，推了推眼镜）

你们有没有想过一个问题——

你妈在厨房喊你吃饭，你隔着三堵墙、两扇门、一条走廊，还是听得清清楚楚。

但如果你妈在太空里喊你吃饭——
你什么都听不到。

不是因为你妈在太空里不会喊你吃饭。
（笑）
而是因为太空是真空，声音传不过去。

你看，你妈在地球上喊你吃饭管用，是因为空气帮你妈传了话。
你妈到了太空喊你吃饭不管用，是因为没有空气当"快递员"。

（语速平稳，但眼神发亮）

声音需要介质来传播。
固体、液体、气体都可以当介质。
但真空——不行。

所以物理告诉你一个道理：
珍惜你妈在地球上喊你吃饭的日子。
因为如果她在太空里喊你，你不但吃不上饭，连听都听不见。
```
