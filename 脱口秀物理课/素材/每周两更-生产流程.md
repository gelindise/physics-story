# ⏰ 冷面物理 · 每周两更生产流程

> 发布时间：周三正片 + 周日轻量内容

---

## 每周工作流

### 周一 · 选题+素材提取

```
□ 确认本周知识点（对照教学进度）
□ 从「物理解码者」对应章节提取素材
□ 收集生活中的相关段子素材
  输出：一页笔记，包含 3-5 个段子角度
```

### 周二 · 写稿+打磨

```
□ 按逐字稿模板写初稿
□ 检查：每个段子的预期违背结构是否完整
□ 检查：知识点的准确性
□ 打磨节奏：朗读一遍，调整停顿和语气标记
  输出：周三正片定稿
```

### 周三 · 发布正片

```
□ 最终排版（公众号编辑器中调整）
□ 添加封面图和标题
□ 发布 EP 正片
  输出：公众号文章
```

### 周四 · 收集反馈

```
□ 查看留言和互动数据
□ 记录哪些梗效果好的
□ 记录知识点是否需要补充
  输出：反馈笔记
```

### 周五-周六 · 创作周日内容

```
□ 写周日轻量篇（互动挑战/段子合集/答疑）
□ 同时准备下周正片的初稿
  输出：周日稿 + 下周初稿
```

### 周日 · 发布轻量内容

```
□ 发布互动/答疑/段子合集
  输出：公众号文章
```

## 双周排期法

| 周 | 周三正片 | 周日轻量 | 进度追赶 |
|----|---------|---------|---------|
| 第 1 周 | EP 02 声音 | 互动：测测你的听力 | 建立模板 |
| 第 2 周 | EP 03 温度·熔化凝固 | 互动：温度猜猜猜 | 调整节奏 |
| 第 3 周 | EP 04 汽化液化升华凝华 | 互动：物态变化接龙 | 储备下一批 |
| 第 4 周 | EP 05 光的反射 | 互动：镜子迷宫 | 储备下一批 |
| 第 5 周 | EP 06 光的折射·透镜 | 互动：透镜实验猜猜看 | 开始写第 2 批 |

**批量创作策略**：一次写 2 周以上的稿子，形成储备后再上线，避免赶稿。

---

## 内容储备目标

```
第一批（上线前储备）：EP 00 + EP 01 + EP 02 + EP 03 = 4 期
   → 足够第一个月发布，从容追赶

第二批（上线后储备）：EP 04 + EP 05 + EP 06 + EP 07 = 4 期
   → 第二个月弹药

持续滚动：保持 4 期以上的提前量
```

## 「冷面物理」风格 check list

发布前自检：

- [ ] 语气够沉稳吗？（不刻意搞笑，不夸张尖叫）
- [ ] 内心热情传达到了吗？（读到精彩处有没有感染力？）
- [ ] 段子有"预期违背"的结构吗？
- [ ] 物理知识准确吗？（有不确定的查教材确认）
- [ ] 每段不超过 4 行吗？
- [ ] 画面感够强吗？（读者能"看见"场景吗？）
- [ ] 节奏标记（停顿/笑）够自然吗？
- [ ] 有 call-back 上一期吗？
- [ ] 有推荐「物理解码者」对应章节吗？
- [ ] 互动挑战有梗选项吗？
- [ ] 本期金句能传播吗？
